# Приложение для расписания учителя.

## О проекте:
- PET-проект приложения для расписания и взаимодействия учителя со своими классами. Приложение, к сожалению, было заброшено, но была реализована большая часть функционала. Такая как:
    Личный кабинет( регистарция/логин )
    Создание и привязка классов/предметов
    Загрузка файлов и их отображение у участников
    API под будущее расширение, но не реализованное из-за фронта

## Основная сложность.
- В освоинии абсолютно незнакомых технологии.
